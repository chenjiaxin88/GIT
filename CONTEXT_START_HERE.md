# Long-Term Context Entry

This repository is intended to work as a local long-term context base for Chen Jiaxin.

Detailed personal context is stored locally in `private-context/` and is intentionally ignored by Git. Do not push that folder to GitHub unless the owner explicitly decides the repository is private and wants those files tracked.

Recommended load order for future AI/Codex sessions:

1. Always start with `private-context/00-boot-card.md`.
2. Then follow `private-context/00-context-protocol.md`.
3. Only load `01` to `05` when the current task needs deeper personal, work, business, or planning context.
4. Use `conversation-index/` only for retrieval, not as startup context.

If the private folder is missing, ask the owner to provide the latest context export or summary before making personal, career, or project-level judgments.
